# Library Management System

This is a simple desktop application built using Python and Tkinter to manage books in a library.

## Features
- Add new books
- Remove books
- Display available books
- Simple graphical user interface

##  Technologies Used
- Python
- Tkinter

##  How to Run

1. Install Python
2. Download the project
3. Run the file

python lms.py
